qu'est-ce que c'est ?

rpmrebuild est un logiciel qui permet de construire des packages rpm depuis des packages rpm installés.
Dans une utilisation de base, il ne demande aucune connaissance sur la fabrication des rpm.
C'est l'equivalent du logiciel dpkg-repack sur debian.

plateformes supportees

toute distribution linux utilisant les packages rpm :
  • redhat
  • fedora
  • whitebox
  • mandrake
  • suse
  • and more

contribution à rpmrebuild

Il y a de nombreuses façons de contribuer au projet, même si vous n'êtes pas un développeur :

bug report

Si vous trouvez un bug, ou si rpmrebuild ne marche pas comme attendu, addressez-vous au support

feature request

Si vous avez besoin de nouvelles fonctionnalités, adressez-vous aussi au support.

translation

La version 2.2.x apporte l'internationalisation des messages. Les traductions en français et anglais sont fournies par l'équipe de developpemment, mais il est assez simple de rajouter une nouvelle traduction. Il suffit de rajouter un fichier dans l'arborescence /usr/lib/rpmrebuild/locale/ (par exemple une traduction en allemand serait dans le fichier /usr/lib/rpmrebuild/locale/de/rpmrebuild.lang). Envoyez-nous vos traductions, elles seront intégrées dans les versions suivantes.

plugins

Si vous avez écrit des plugins, envoyez-les nous, afin de pouvoir les partager avec les autres utilisateurs.
